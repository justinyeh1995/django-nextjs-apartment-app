# Demo

## Installation
```bash
git clone https://github.com/<>
cd <>
docker-compose build -t demo-full-stack
docker-compose up --rm
```

## Build Backend locally

```bash
python3 -m venv demo-venv
source demo-venv/bin/activate
```

```bash
python3 -m pip install -r requirements.txt
```
